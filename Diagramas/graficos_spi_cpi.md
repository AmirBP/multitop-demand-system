## Código Python para Generar Gráficos Reales (Opcional)

Si deseas generar estos gráficos con matplotlib:

```python
import matplotlib.pyplot as plt
import numpy as np

# Datos Sprint 1
semanas_s1 = [1, 2, 3, 4]
pv_s1 = [4500, 9000, 13500, 18000]
ac_s1 = [4600, 9100, 13400, 18100]
ev_s1 = [4500, 9000, 13500, 18000]

# Crear gráfico
fig, ax = plt.subplots(figsize=(10, 6))
ax.plot(semanas_s1, pv_s1, marker='o', label='PV (Planificado)', linewidth=2)
ax.plot(semanas_s1, ac_s1, marker='^', label='AC (Real)', linewidth=2)
ax.plot(semanas_s1, ev_s1, marker='s', label='EV (Ganado)', linewidth=2)

ax.set_xlabel('Semanas')
ax.set_ylabel('Valor (S/)')
ax.set_title('Curva de Valor Ganado - Sprint 1')
ax.legend()
ax.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig('evm_sprint1.png', dpi=300)
plt.show()
```
